METHOD — how this list was built
  • Source: all bonded tokens on pump.tires (PulseChain launchpad) — ~950 candidates.
  • Paid filter: token has a paid DexScreener banner (info.header) — actively enforced by both the 🚀 button and refresh.py; tokens that lose the banner fall off.
  • Mcap filter: real circulating mcap < $500 — computed as (total_supply − LP-locked reserve) × price.
  • LP filter: ≤ 1 token-token LP per DexScreener /token-pairs (excludes wide LP networks).
  • Vibes filter: subjective "good meme" pass — manual, not scored.
  • Auto-check: 🚀 button re-verifies every row via DexScreener (batched), moves fallers into FELL OFF and resurrects recovered ones back to FULL LIST. Tokens in SAVED / PICKS get fresh data but never move.
  • Discovery: python3 refresh.py locally — refreshes the bonded pool, verifies all tokens, and pulls in any new bonded tokens that meet criteria. Add --commit to push.
  • Source of truth: master.json (gitignored) — every token ever seen; sections only move, tokens never get deleted.
